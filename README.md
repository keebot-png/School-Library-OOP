# 📖 School Library <a name="about-project"></a>

- This project makes the use of object oriented programming by storing information in classes, assigning them variables as well as methods to carry out certain actions. This database it specifically tailored to that of a school library.

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Has a file for each class which stores information about certain people within the school setting**
- **Uses composition and design patterns to decorate a person class, specifically the names of people**
- **Implement the different associations between classes**
- **Execute a script that stores input variables (gets.chomp) into instantiated classes**

## 💻 Getting Started <a name="getting-started"></a>

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/keebot-png/School-Library-OOP.git
  cd School-Library-OOP
```

### Install

Install this project with:

```sh
  gem install rubocop
  bundle init
  bundle install
```

### Usage

```sh
  rails server
```
